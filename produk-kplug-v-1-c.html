K-PLUG

Alat Praktis untuk Latihan Musik dan Rekaman Langsung di Smartphone

Fungsi dan Kegunaan

  • Silent Practicer: Latihan alat musik secara diam-diam tanpa mengganggu lingkungan sekitar, sambil tetap bisa merekam.
  • Mixer Alat Musik untuk Smartphone: Menghubungkan alat musik dengan smartphone untuk menggabungkan audio.
  • Simple dan Ekonomis: Mudah digunakan dan terjangkau, cocok untuk musisi yang ingin merekam secara cepat.
  • Mendukung Hampir Semua Smartphone: Kompatibel dengan berbagai merek smartphone.
  • Mendukung Live Streaming: Dapat digunakan untuk streaming langsung ke platform sosial media dengan suara berkualitas.

Fitur Utama - Versi 1.0

  • Personal Silent Practicer: Dilengkapi dengan input pasif untuk latihan tanpa gangguan.
  • Sebagai Audio Link: Digunakan untuk latihan sekaligus merekam video dengan cepat dan mudah.
  • Dukungan untuk Live Streaming: Langsung live streaming ke media sosial tanpa alat tambahan.
  • Fitur Loopback: Merekam video dan suara secara bersamaan dari sumber yang sama.
  • Autovoltage 5-9 Volt: Penyesuaian otomatis voltase untuk berbagai adaptor charger dan powerbank.
  • Dukungan untuk Amplifier Eksternal: Dapat digunakan sebagai input untuk amplifier eksternal seperti kamera DSLR atau mirrorless.

Cara Penggunaan

  1. Download aplikasi "Open Camera" dari Play Store dan atur sumber audio di aplikasi tersebut.
  2. Pilih sumber audio eksternal dari aplikasi yang akan digunakan (misalnya dari K-PLUG).
  3. Sesuaikan level input suara di aplikasi agar tidak terlalu kecil atau besar.
  4. Mulai merekam video dengan audio dari alat musik yang terhubung.
  5. Periksa grafik audio dari input untuk memastikan hasil rekaman optimal.
  6. Disarankan menggunakan power supply eksternal untuk kestabilan daya.

Tanya Jawab

  • Apakah K-PLUG bisa dijadikan soundcard untuk merekam di laptop?
    Tidak, K-PLUG hanya khusus untuk smartphone.
  • Apakah ada garansi?
    Garansi tersedia jika barang diterima dalam kondisi baik dan selama pemakaian awal.
  • Hasil rekaman ada noise?
    Cek adaptor power supply, beberapa jenis charger mungkin tidak cocok. Disarankan menggunakan adaptor stompbox 9V atau powerbank berkualitas.
  • Hasil rekaman kecil?
    Cek pengaturan input dan sensitivitas HP. Beberapa HP mungkin perlu penyesuaian sensitivitas pada aplikasi perekam.
  • Apakah bisa untuk rekam mic?
    K-PLUG tipe 1c mendukung input pasif mono, tetapi tidak mendukung penggunaan mic dinamik atau condenser secara langsung.
